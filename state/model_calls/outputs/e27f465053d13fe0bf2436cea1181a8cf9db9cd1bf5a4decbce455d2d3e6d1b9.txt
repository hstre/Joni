["Lexical Consensus is an experimental framework for studying grounded word learning using frozen DINOv2 visual embeddings, Carroll-style nonce words, and interpretable lexical learners plus linear baselines.","A perceptual-coherence gradient governs learnability: native categories are easiest, coherent overextensions remain learnable, mid-range disjunctive concepts degrade, and far-disjunctive concepts approach chance.","Perceptual distance, not semantic distance, predicts acquisition accuracy (partial R^2 = 0.245, p < 1e-7), while semantic distance adds no significant explanatory power (partial R^2 = 0.002, p = 0.660).","Naming and retrieval are distinct, with exemplar-based mechanisms outperforming centroid prototypes in label-to-image retrieval, exposing a memory-fidelity dimension separate from naming accuracy."]
